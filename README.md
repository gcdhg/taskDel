# Task job

## Run locally on your machine

You need to have installed Node and NPM.

Clone this repository

```bash
git clone https://github.com/gcdhg/taskDel.git
```

Install dependencies

```bash
npm install
```

Start NodeJS server at http://localhost:3000

```bash
npm start
```

For Nodemon usage

```bash
npm run dev
```

run tests

```shell
npm test
```
